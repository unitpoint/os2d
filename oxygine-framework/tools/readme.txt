how to use "oxyresbuild.py":
1. install Python 2.7 from http://python.org/download/
2. install Python Imaging Library (for Python27) from http://www.pythonware.com/products/pil/  
3. add to your system environment variable 'PATH' path to Python folder
4. run install.py. It would add current path of Oxygine to System global variable
